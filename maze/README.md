# Maze Game

Ini adalah game labirin sederhana yang dibuat dengan HTML, CSS, dan JavaScript.

## Teknologi yang Digunakan

*   **HTML**: Bahasa markup standar untuk membuat halaman web.
*   **CSS**: Bahasa stylesheet yang digunakan untuk mendeskripsikan presentasi dokumen yang ditulis dalam bahasa markup.
*   **JavaScript**: Bahasa pemrograman yang sesuai dengan spesifikasi ECMAScript.

## Cara Bermain

Buka file `index.html` di browser web Anda.
