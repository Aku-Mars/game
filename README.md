# Koleksi Game Sederhana

Repositori ini berisi kumpulan game berbasis web sederhana yang dibuat untuk tujuan demonstrasi dan pembelajaran.

## Daftar Game

*   **Epic Dragon**: RPG petualangan berbasis teks di mana Anda memilih kelas, melawan monster, mengumpulkan jarahan, dan menaklukkan tahapan yang menantang.
*   **Maze**: Game labirin sederhana di mana tujuannya adalah menemukan jalan keluar.
*   **Tebak Angka**: Game tebak angka klasik di mana pemain harus menebak angka rahasia dalam jumlah percobaan yang terbatas.

## Teknologi yang Digunakan

Setiap game dibangun menggunakan teknologi yang berbeda:

*   **Epic Dragon**: React, TypeScript, Vite
*   **Maze**: HTML, CSS, JavaScript (Vanilla)
*   **Tebak Angka**: HTML, CSS, JavaScript (Vanilla)

## Pengaturan Umum

Untuk game yang menggunakan `package.json` (seperti **Epic Dragon**), navigasikan ke subdirektori game dan jalankan:

```bash
npm install
```

Kemudian, untuk memulai server pengembangan, jalankan:

```bash
npm run dev
```

Untuk game yang hanya terdiri dari file HTML, CSS, and JavaScript, cukup buka file `index.html` di browser web Anda.
